## 🚀 Mission‑critical projects

### 🧠 Predictive GPU Memory Defragmenter

Transformer-based engine that models GPU memory fragmentation and proactively defragments training workloads.

- Fewer out-of-memory failures and higher GPU utilization on NVIDIA RTX-class hardware.  
- Stack: PyTorch, custom GPU memory profiling, sequence modeling.

### 🛰️ Apex‑Aegis Tactical Suite

Physics-informed neural network engine that simulates multi‑physics trajectories under strict safety constraints.

- Risk‑aware, low‑latency trajectory predictions for aerospace scenarios.  
- Stack: PINNs, RK4 integrator, atmospheric and dynamics modeling.

### 📡 CommandX — Satellite Mission Control

End-to-end mission-control stack unifying orbital physics, autonomous guidance/control, and telemetry visualization.

- Real-time streaming of orbital elements, anomaly flags, and system health for satellites.  
- Stack: Python, orbital mechanics libraries, real‑time dashboards and telemetry.

### 🔐 PulseNet — Secure Telemetry & Maintenance

Security-first predictive maintenance pipeline for sensor fleets and industrial telemetry.

- Early anomaly detection with encrypted storage and auditable access patterns.  
- Stack: time‑series ML, cryptography, secure cloud infrastructure.
